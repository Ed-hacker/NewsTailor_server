```java
// JWT 인증 필터 핵심 로직
@Override
protected void doFilterInternal(HttpServletRequest request,
                              HttpServletResponse response,
                              FilterChain filterChain) {
    String authHeader = request.getHeader("Authorization");
    if (authHeader != null && authHeader.startsWith("Bearer ")) {
        String jwt = authHeader.substring(7);
        String username = jwtUtil.extractUsername(jwt);

        if (username != null && SecurityContextHolder.getContext()
            .getAuthentication() == null) {
            UserDetails userDetails = userDetailsService
                .loadUserByUsername(username);

            if (jwtUtil.isTokenValid(jwt, userDetails)) {
                UsernamePasswordAuthenticationToken authToken =
                    new UsernamePasswordAuthenticationToken(
                        userDetails, null, userDetails.getAuthorities());
                SecurityContextHolder.getContext().setAuthentication(authToken);
            }
        }
    }
    filterChain.doFilter(request, response);
}

// Spring Security 설정
@Bean
public SecurityFilterChain filterChain(HttpSecurity http) throws Exception {
    return http
        .csrf(csrf -> csrf.disable())
        .sessionManagement(session ->
            session.sessionCreationPolicy(SessionCreationPolicy.STATELESS))
        .authorizeHttpRequests(auth -> auth
            .requestMatchers("/api/signup", "/api/login").permitAll()
            .anyRequest().authenticated())
        .addFilterBefore(jwtAuthenticationFilter,
            UsernamePasswordAuthenticationFilter.class)
        .build();
}
```

## 📋 워드 작성 시 활용할 구체적 내용

### OpenAI → Gemini 마이그레이션 섹션

**변경 배경**:
```
기존 OpenAI GPT API는 유료 서비스로 학생 프로젝트에 경제적 부담이 있었으나,
Google Gemini API는 대학생에게 무료로 제공되어 프로젝트의 지속가능성을 크게 향상시켰습니다.
```

**기술적 차이점 표**:
| 구분 | OpenAI | Gemini |
|------|--------|--------|
| 인증 방식 | Bearer Token (헤더) | API Key (URL 파라미터) |
| 요청 구조 | `{messages: [...]}` | `{contents: [...]}` |
| 응답 구조 | `choices[0].message.content` | `candidates[0].content.parts[0].text` |
| 비용 | 유료 | 무료 (대학생) |

**마이그레이션 결과**:
- 운영비 100% 절감
- 한국어 요약 성능 동등 수준 유지
- 기존 인터페이스 호환성 보장

### 주요 구현 기능 섹션

**완료된 핵심 기능들**:
1. ✅ JWT 기반 인증/인가 시스템 (100%)
2. ✅ 네이버 뉴스 랭킹 자동 크롤링 (100%)
3. ✅ Gemini AI 요약 서비스 (100%)
4. ✅ 사용자 관리 및 관심사 설정 (100%)

**기술적 특징**:

### 코드 설명 템플릿

**크롤링 시스템 설명**:
```
네이버 뉴스 랭킹 크롤링 시스템은 Spring의 @Scheduled 애노테이션을 활용하여
매 정시 5분마다 자동으로 실행됩니다. Jsoup 라이브러리를 사용해 HTML을 파싱하고,
CSS 선택자를 통해 정확한 뉴스 데이터를 추출합니다.
각 섹션별로 예외 처리를 구현하여 일부 실패 시에도 다른 섹션의 크롤링이
계속 진행되도록 안정성을 확보했습니다.
```

**JWT 인증 시스템 설명**:
```
JWT 기반 인증 시스템은 무상태(Stateless) 방식으로 구현되어 서버 확장성을 보장합니다.
OncePerRequestFilter를 상속받은 커스텀 필터를 통해 모든 요청의 Authorization 헤더를 검증하고,
유효한 토큰인 경우 SecurityContext에 인증 정보를 설정합니다.
HS512 알고리즘을 사용한 강력한 서명으로 보안성을 확보했습니다.
```
